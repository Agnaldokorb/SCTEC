Repositorio de aulas

Front-end (HTML - CSS - JAVASCRIPT)

Este repositório reúne os conteúdos, exercícios e projetos desenvolvidos durante o curso de Desenvolvimento Front-end do SENAI, ministrado pelo professor Eduardo Worrel. Ao longo das aulas, são abordados os fundamentos de HTML, CSS e JavaScript, com foco na construção de interfaces web, estilização de páginas e desenvolvimento de funcionalidades interativas. O objetivo deste repositório é documentar minha evolução, consolidar os conhecimentos adquiridos e servir como referência para estudos e futuros projetos.
